VS Code Remote SSHиҝһжҺҘеӨұиҙҘзҡ„й—®йўҳи§ЈеҶі

жң¬ж–ҮжЎЈй’ҲеҜ№VS Codeдёӯзҡ„Remote SSHжҸ’д»¶еңЁе°қиҜ•иҝһжҺҘиҝңзЁӢжңҚеҠЎеҷЁж—¶еҮәзҺ°вҖқFailed to parse remote port from server outputвҖқй”ҷиҜҜзҡ„жғ…еҶөжҸҗдҫӣи§ЈеҶіж–№жЎҲгҖӮдҪңиҖ…еңЁз»ҸиҝҮдёҖзі»еҲ—зҡ„жҺ’жҹҘе’Ңе°қиҜ•еҗҺпјҢжңҖз»ҲжүҫеҲ°дәҶи§ЈеҶізҡ„ж–№жі•гҖӮ

й—®йўҳжҸҸиҝ°

еңЁдҪҝз”ЁVS Codeзҡ„Remote SSHжҸ’д»¶е°қиҜ•иҝһжҺҘиҝңзЁӢжңҚеҠЎеҷЁж—¶пјҢйҒҮеҲ°дәҶй”ҷиҜҜжҸҗзӨәвҖқFailed to parse remote port from server outputвҖқгҖӮжӯӨй”ҷиҜҜжҸҗзӨәеҸҜиғҪжҳҜз”ұдәҺVS CodeдёҚиғҪжӯЈзЎ®ең°д»ҺSSHжңҚеҠЎеҷЁзҡ„иҫ“еҮәдёӯи§ЈжһҗеҮәиҝңзЁӢз«ҜеҸЈгҖӮ

е°қиҜ•зҡ„и§ЈеҶіж–№жі•

  1. жЈҖжҹҘSSHй…ҚзҪ®ж–Үд»¶
  2. жӣҙж–°VS Codeе’ҢRemote SSHжү©еұ•
  3. жүӢеҠЁSSHиҝһжҺҘ
  4. жЈҖжҹҘиҝңзЁӢжңҚеҠЎеҷЁзҡ„зҠ¶жҖҒ
  5. йҮҚеҗҜVS Code

д»ҘдёҠеёёи§Ғзҡ„и§ЈеҶіж–№жі•йғҪжңӘиғҪи§ЈеҶій—®йўҳгҖӮ

жҲҗеҠҹзҡ„и§ЈеҶіж–№жЎҲ

жңҖз»ҲпјҢдҪңиҖ…е°қиҜ•дәҶеҸ–ж¶ҲеӢҫйҖүVS Codeи®ҫзҪ®дёӯзҡ„Remote.SSH: Use Local ServerйҖүйЎ№пјҢжҲҗеҠҹиҝһжҺҘеҲ°дәҶиҝңзЁӢжңҚеҠЎеҷЁгҖӮеҪ“иҜҘйҖүйЎ№иў«йҖүдёӯпјҲй»ҳи®Өпјүж—¶пјҢVS CodeдјҡеңЁжң¬ең°жңәеҷЁдёҠеҗҜеҠЁдёҖдёӘжңҚеҠЎеҷЁпјҢз„¶еҗҺйҖҡиҝҮиҜҘжң¬ең°жңҚеҠЎеҷЁиҝһжҺҘеҲ°иҝңзЁӢSSHжңҚеҠЎеҷЁгҖӮеҪ“еҸ–ж¶ҲеӢҫйҖүжӯӨйҖүйЎ№ж—¶пјҢVS CodeдјҡзӣҙжҺҘиҝһжҺҘеҲ°иҝңзЁӢSSHжңҚеҠЎеҷЁпјҢиҖҢдёҚйҖҡиҝҮжң¬ең°жңҚеҠЎеҷЁгҖӮ

и§ЈеҶіжӯҘйӘӨ

  1. жү“ејҖVS CodeгҖӮ
  2. еңЁе·Ұдҫ§зҡ„жҙ»еҠЁж ҸзӮ№еҮ»йҪҝиҪ®еӣҫж Үжү“ејҖи®ҫзҪ®гҖӮ
  3. еңЁи®ҫзҪ®жҗңзҙўжЎҶдёӯиҫ“е…ҘRemote.SSH: Use Local ServerгҖӮ
  4. еҸ–ж¶ҲйҖүдёӯеҮәзҺ°зҡ„Remote.SSH: Use Local ServerеӨҚйҖүжЎҶгҖӮ

жіЁж„ҸдәӢйЎ№

иҷҪз„¶зӣҙжҺҘиҝһжҺҘеҲ°иҝңзЁӢSSHжңҚеҠЎеҷЁеҸҜд»Ҙи§ЈеҶіжҹҗдәӣиҝһжҺҘй—®йўҳпјҢдҪҶз”ұдәҺжІЎжңүеҲ©з”ЁеҲ°жң¬ең°жңҚеҠЎеҷЁзҡ„дјҳеҠҝпјҢеҸҜиғҪдјҡеҜјиҮҙVS Codeзҡ„жҖ§иғҪзЁҚжңүдёӢйҷҚгҖӮдҪҶеҸӘиҰҒжІЎжңүйҒҮеҲ°жҖ§иғҪй—®йўҳпјҢиҝҷдёӘи®ҫзҪ®е°ұдёҚйңҖиҰҒиҝҮдәҺжӢ…еҝғгҖӮ

з»“и®ә

еҰӮжһңдҪ еңЁдҪҝз”ЁVS Codeзҡ„Remote SSHжҸ’д»¶иҝһжҺҘиҝңзЁӢжңҚеҠЎеҷЁж—¶йҒҮеҲ°дәҶзұ»дјјзҡ„й—®йўҳпјҢдҪ д№ҹеҸҜд»ҘиҜ•иҜ•иҝҷдёӘж–№жі•пјҢеёҢжңӣиҝҷдёӘи§ЈеҶіж–№жЎҲиғҪеё®еҠ©еҲ°дҪ гҖӮ
