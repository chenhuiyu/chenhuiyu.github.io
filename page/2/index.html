python多线程存在的问题

GIL全称为Global Interpreter Lock,即为全局解释所。

在python多线程下，每个线程的执行方式如下：

  • 获取GIL
  • 执行对应线程的代码
  • 释放GIL

某个线程想要执行，必须先拿到GIL，一个python进程中GIL只有一个，GIL的存在使得Python无论是在单核还是多核的条件下，同一时刻只能运行一个线程，导致python多线程无法发挥多核并行的优势。

python多进程的实现

进程是系统进行资源分配和调度的独立单位，各个进程之间的数据是无法共享的，多个进程无法共享一个全局变量。

Python内置的multiprocessing是用来实现多进程的库。multiprocessing提供了一系列组件，如Process（进程）、Queue（队列）、Semaphore（信号量）、Pipe（管道）、Lock（锁）、Pool（进程池）

Process类

在multiprocessing中，每个进程都用一个Process类来表示

Process([group [, target [, name [, args [, kwargs]]]]])

  • target 表示调用对象，可以传入方法的名字
  • args表示被调用对象位置参数元组，比如target是函数func，他有两个参数m,n，那么args就传入[m,n]即可
  • kwargs表示调用对象的字典
  • name是别名，相当于给这个进程取个名字
  • group分组

线性回归

[toc]

线性回归输出是一个连续值，因此适用于回归问题。softmax回归则适用于分类问题。线性回归和softmax回归都是单层神经网络，以线性回归为例，介绍大多数深度学习模型的基本要素和表示方法。

1 线性回归的基本要素

我们以一个简单的房屋价格预测作为例子来解释线性回归的基本要素。这个应用的目标是预测一栋房子的售出价格（元）。为了简单起见，这里我们假设价格只取决于房屋状况的两个因素，即面积（平方米）和房龄（年）。接下来我们希望探索价格与这两个因素的具体关系。

1.1 模型

设房屋的面积为\(x1\)，房龄为\(x2\)，售出价格为\(y\)。我们需要建立基于输入\(x1\)和\(x2\)来计算输出\(y\)的表达式，也就是模型（model）。线性回归假设输出与各个输入之间是线性关系：

\[\hat{y}=x_1w_1+x_2w_2+b\]

其中\(w_1\)和\(w_2\)是权重（weight），\(b\)是偏差（bias），且均为标量。它们是线性回归模型的参数（parameter）。模型输出\(\hat{y}\)是线性回归对真实价格\(y\)的预测或估计。我们通常允许它们之间有一定误差。

基于SRGAN的超分辨率重建

[toc]

1. 简介

1.1 超分辨率重建问题

​ 超分辨率重建是指将低分辨率图像(LR)恢复成高分辨率图像(HR)的过程，一张被重建的高分辨率图像被称为超分辨率图像或SR图像。超分辨率重建仍旧是计算机视觉领域一个富有挑战性的问题。首先，超分辨率重建问题是一个不适定问题，对于同一个低分辨率图像，存在多个解决方案，而不是一个唯一的解决方案。 为了限制解决方案空间，通常需要可靠的先验信息。此外，超分辨率重建问题的复杂性随着放大倍数的增加而增加。 在较高的放大系数下，丢失场景细节的恢复变得更加复杂。如何用一大较大的放大系数的情况下仍能恢复画质细节成为研究的难点。目前的大部分研究都聚焦于最小化重建均方误差。以此作为优化的结果会带来很高的PSNR(峰值信噪比)，但是通常的结果是缺乏高频细节以及在感知上不与原图匹配。

定量指标

  • PSNR = 峰值信噪比 PSNR是信号最大可能功率和影响它的表示精度的破坏性噪声功率的比值

  • SSIM = 结构相似性指数 SSIM测量两个相似图像之间的感知差异

1.2 生成对抗网络

​ 生成对抗网络（GAN）是一种基于博弈论方法的深度神经网络体系结构，其中模型的两个组成部分，即生成器和判别器。生成器尝试捕获真实示例的分布，以生成新的数据示例。判别器通常是二分类器，它尽可能准确地将生成器产生的的数据与真实的样本区分开。 GAN的优化是一个最小最大(minimax)优化问题。该优化终止于一个鞍点，该鞍点对于生成器而言是最小值，而对于鉴别器而言是最大值。也就是说，优化目标是达到Nash均衡。然后，可以认为生成器已经捕获了真实样本的概率分布。 通过这种方式，生成器学习生成更好，更逼真的图像。

​ GAN适用于超分辨率问题：生成器创建SR图像，使鉴别器无法将其识别为真实HR图像或人为的超分辨输出。 以这种方式，产生具有更好的感知质量的HR图像。

2. 损失函数

SRGAN将损失函数由MSE改为感知损失函数。感知损失是内容损失和对抗性损失的加权总和。

\(l^{SR}=l^{SR}_X+10^{-3}l^{SR}\_{Gen}\)

其中\(l^{SR}\)叫做感知损失，\(l^{SR}_X\)是内容损失，\(10^{-3}l^{SR}_{Gen}\)是对抗损失

2.1 内容损失

内容损失有两种类型：

2.1.1 Pixel-wise MSE损失

​ 逐个像素的MSE损失（Pixel-wise MSE）是真实图像中每个像素与生成图像中一个像素之间的平方误差。Pixel-wise MSE强调的是两幅图像之间每个对应像素的匹配，这与人眼的感知结果有所区别。通过pixel-wise loss训练的图片通常会较为平滑，缺少高频信息。即使输出图片具有较高的PSNR，视觉效果也并没有很突出。

\[l^{SR}_{MSE}=\frac{1}{r^2WH}\sum\limits^{rW}_{x=1}\sum\limits^{rH}_{y=1}(I^{HR}_{x,y}-G_{\theta_{G}}(I^{LR})_{x,y})^2\]

  • \(W,H\)：图像的宽度和高度
  • \(r\)：下采样因子
  • \(G_{\theta_{G}}\)：生成器，一个前馈CNN网络
  • \(\theta_{G}={W_{1:L;b{1:L}}}\):L层神经网络的参数，包括Weight和bias
  • \(G_{\theta_{G}}(I^{LR})\)：由生成器生成的高分辨率图像，

1. 简介

​ GAN包含两个模型：生成器和判别器。这两个模型通常由神经网络来实现，生成器尝试捕获真实示例的分布，以生成新的数据示例。判别器通常是二分类器，它尽可能准确地将生成器产生的的数据与真实的样本区分开。 GAN的优化是一个最小最大(minimax)优化问题。该优化终止于一个鞍点，该鞍点对于生成器而言是最小值，而对于鉴别器而言是最大值。也就是说，优化目标是达到Nash平衡[1]。然后，可以认为生成器已经捕获了真实样本的概率分布。

​ 先前的一些工作采用了使两个神经网络相互竞争的概念。GAN属于生成算法。生成算法和判别算法是两类机器学习算法。如果机器学习算法是基于观测数据的完全概率模型，则该算法是生成性的。生成算法由于其广泛的实际应用而变得越来越流行和重要。

4.1 静态网页和动态网页

​ 最基本的HTML代码，我们将其保存为一个.html文件，然后把它放在某台具有固定公网IP的主机上，主机上装上Apache或Nginx等服务器，这样这台主机就可以作为服务器了，其他人可以通过访问服务器看到这个页面，这样就搭建了一个最简单的网站。

​ 这种HTML代码编写的网页叫静态网页，其文字、图片均是通过HTML代码指定。静态网页加载速度快，编写简单，但是可维护性差，不能根据URL灵活多变的展示内容。

​ 动态网页可以动态解析URL中的参数的变化，关联数据库并动态呈现不同的页面内容。现在大多数网页都是动态网页，它们不再是一个简单的HTML，而是可能有JSP、PHP、python等语言编写的。

​ 动态网页还可以实现用户登录和注册功能，需要通过Session和Cookies共同作用
